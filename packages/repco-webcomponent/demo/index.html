Repco Post Element

The repco-post-element custom element is a reusable and customizable component that displays post data fetched from a GraphQL API.

Usage

To use the repco-post-element custom element, you need to include the following code in your HTML file: 

            <repco-post-element></repco-post-element>

Attributes

The following are the attributes of the repco-post-element custom element:

  • theme: A string attribute that sets the visual theme of the post element. It can be either "light" or "dark". The default value is "light".
  • layout: A string attribute that sets the layout of the post elements. It can be either "column" or "horizontal". The default value is "column".
  • count: A number attribute that sets the number of post elements to be displayed. The default value is 10.
  • _defaultThumbnail: A string attribute that sets the default thumbnail image to be displayed if a post doesn't have a media asset of type 'image'. The default value is an empty string.
  • endpoint: A string attribute that sets the endpoint for the GraphQL API to fetch the post data. The default value is 'https://node1.repco.openaudiosearch.org/graphql'.
  • query: A string attribute that sets the GraphQL query used to fetch the post data from the API. The default query returns the title, content, uid, and media assets for all content items.
